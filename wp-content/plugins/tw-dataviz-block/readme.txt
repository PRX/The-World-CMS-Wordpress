== Description ==

This plugin configures a new variation for the `core/embed` block to properly embed DataViz repsonsive embeds.
Ensures responsive iframe resize script is loaded and iframe is styled responsiveliy in editor.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/tw-dataviz-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin by adding `'wp-content/plugins/tw-dataviz-block/tw-dataviz-block.php'` to the-world-site-config > configs > global > global-plugins.php in mu-plugins.
